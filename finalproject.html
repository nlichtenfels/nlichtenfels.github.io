My Final Project

Me and my partners final project for this course was the recreation of a popular mobile game, Flappy Bird.
Flappy Bird is an endless sidescroller where the point of the game is to jump in between sets of randomly
generated pipes, gaining a point for every pipe you jump in between, and losing if you hit any pipe. The
controls for the game are simple, the only control is left clicking to make the bird jump. Pressing start
game or try again will start the game, and pressing the quit game will take you back to the main menu.
Good luck and have fun trying to get new highscores!
